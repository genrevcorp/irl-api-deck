01 — The problem01 — 問題

Software runs every business. It just can't see. 軟體管理著每一家企業，卻看不見現場。

The ERP knows the orders, the bookings, the inventory. It doesn't know whether table 4 is actually cleared, court 2 is actually occupied, or bird #1042 stopped eating this morning. ERP 知道訂單、預約、庫存——但它不知道 4 號桌到底收乾淨了沒、2 號球場現在有沒有人、1042 號雞今天早上是不是不吃了。

GAP·01

The last 10 meters are manual最後十公尺靠人工

The physical world enters software through a human typing — late, wrong, and expensive. Most of it never gets entered at all. 實體世界要進到軟體裡，得靠人打字輸入——慢、錯、貴。而大部分的事，根本沒人輸入。

GAP·02

CV exists — as closed dashboards電腦視覺存在——但都是封閉儀表板

Camera-AI vendors sell finished end-products to enterprises. Your software can't embed their detections. You get another login, not an API. 攝影機 AI 廠商賣的是賣給企業的成品。你的軟體嵌不進它的偵測結果——你得到的是另一組帳號密碼，不是 API。

GAP·03

"Person detected" is worthless「偵測到人」毫無價值

Generic detection is a commodity. What a business bills, schedules and alerts on is vertical meaning: "table ready", "sick bird", "court free". 通用偵測早已是大宗商品。企業真正用來計費、排程、警示的，是垂直產業的意義：「桌面就緒」、「疑似病雞」、「球場空出」。

02 — The product02 — 產品

One webhook. The real world, structured. 一條 webhook，把真實世界結構化。

We don't sell "person detected." We sell the event your vertical actually runs on — delivered signed, versioned, and idempotent, like a Stripe webhook. 我們賣的不是「偵測到人」，而是你的產業真正賴以運作的事件——像 Stripe webhook 一樣：簽章、版本化、冪等送達。

                 "description": "Cough frequency above per-pet baseline." },
  "media":     { "clip_url": "https://…signed…" }
}

Your developers integrate in an afternoon. Your product gains eyes — without you building a single model. 你的工程師一個下午就能串接完成。你的產品從此有了眼睛——而你不用自己訓練任何模型。

03 — Architecture03 — 架構

Gate cheap. Enrich smart. Deliver clean. 便宜地把關，聰明地理解，乾淨地送達。

01 · edge

Always-on trigger永遠在線的觸發器

A $30-class sensor (camera + mmWave) runs small on-device detection — presence, zones, motion, audio. It watches 24/7 for almost nothing. 約 30 美元等級的感測器（攝影機＋毫米波雷達）在裝置端跑輕量偵測——在席、區域、動作、聲音。24 小時監看，成本趨近於零。

COST ≈ 0 · RECALL FIRST
02 · cloud

VLM verifies the momentVLM 驗證關鍵時刻

Only triggered moments reach a vision-language model, prompt-configured per vertical. A new vertical is a new config pack — not a newly trained model. 只有被觸發的片刻才會送進視覺語言模型，且以提示詞按垂直領域配置。新增一個垂直領域＝一份新設定檔，不是重新訓練模型。

PENNIES / EVENT · BUDGETED
03 · api

Signed webhook out簽章 webhook 送達

Versioned schema, HMAC-signed, retried, idempotent. Plus an actions API back: play a sound, warn an intruder — the loop closes both ways. 版本化 schema、HMAC 簽章、自動重送、冪等。還有反向的 actions API：播放聲音、警示驅離——感知與行動雙向閉環。

irl.event.v1 · 2-WAY

This gate-then-enrich pattern is how Verkada affordably processes ~1 billion hours of video a month — edge detection gates the expensive AI. We apply the same physics to every vertical, as an open API. 這套「先把關、再理解」的模式，正是 Verkada 能以可負擔成本處理每月約 10 億小時影像的原因——邊緣偵測替昂貴的 AI 把關。我們把同樣的物理法則應用到每個垂直領域，並以開放 API 的形式提供。

07 — Business model07 — 商業模式

B2D2B: they keep the customer. We're the senses inside. B2D2B：客戶還是他們的，我們是裡面的感官。

We sell to vertical software companies — ERPs and SaaS that already own the customer relationship. Embedding us is their roadmap's flagship feature; for us it's recurring API revenue without a direct sales army. 我們賣給垂直產業軟體公司——已經握有客戶關係的 ERP 與 SaaS。嵌入我們，是他們產品藍圖上的旗艦功能；對我們而言，則是不需要直銷大軍的經常性 API 收入。

REV·01

Hardware, one-time硬體，一次性收費

Commodity edge sensors at hardware margin. No camera-company ambitions — the device is the delivery vehicle for events. 大宗商品化的邊緣感測器，賺取硬體毛利。我們不做攝影機公司——裝置只是事件的載具。

REV·02

API, per device / monthAPI，按裝置月費

The vertical software pays per connected device for the event stream — priced inside the value of the feature they ship on top of it. 垂直軟體按連線裝置數付月費取得事件串流——定價遠低於他們據此推出的功能所創造的價值。

REV·03

Vertical packs垂直設定包

Premium event types (health-critical detection, compliance) as pack tiers. New packs ship as config — our marginal cost approaches zero. 高階事件類型（健康關鍵偵測、合規監測）以設定包分級收費。新包以設定檔出貨——我們的邊際成本趨近於零。

08 — Defensibility08 — 護城河

Models commoditize. Hardware commoditizes. Verticals don't. 模型會大宗商品化，硬體也會，但垂直領域不會。

M·01

Data flywheel資料飛輪

Every deployed sensor labels its vertical's edge cases. Per-subject baselines (a pet's normal cough rate) can't be bought off the shelf. 每一台部署中的感測器，都在為其垂直領域標註邊角案例。個體基準線（一隻貓「正常」的咳嗽頻率）是買不到的。

M·02

Deployment robustness部署的強健度

The unglamorous 5% — mounting, lighting, dust, flaky Wi-Fi, false-alert tuning. It's where every horizontal platform died. We do it per vertical, once, deeply. 最不光鮮的那 5%——安裝、光線、粉塵、不穩的 Wi-Fi、誤報調校。每個水平平台都死在這裡。我們按垂直領域深耕，一次做透。

M·03

The contract devs build on開發者賴以建構的合約

Once a vertical SaaS ships features on irl.event.v1, switching costs are theirs to fear, not ours. 當垂直 SaaS 基於 irl.event.v1 推出功能後，轉換成本就站在我們這邊了。

M·04

Swappable perception可抽換的感知層

The perception layer is architected to swap — specialized models today, edge VLMs tomorrow. Model progress is our tailwind, not our threat. 感知層天生可抽換——今天用特化模型，明天換邊緣 VLM。模型的進步是我們的順風，不是威脅。

09 — Landscape09 — 競爭地景

Everyone's at the two ends. The middle is open. 所有人都擠在兩端，中間是空的。

Verkada · Ambient.ai · Coram
Own one vertical (security), sell closed end-products direct to enterprises. No embeddable API for other software. They validate the architecture — and leave the API lane empty. 深耕單一垂直領域（安防），把封閉成品直售企業。沒有給其他軟體用的可嵌入 API。他們驗證了這套架構——也把 API 這條賽道空了出來。
Azena † · Azure Percept † · SenseCraft
Horizontal "camera app-store" platforms. Dead or unadopted — platforms-for-everyone serve no one. The graveyard that shaped our vertical-first sequencing. 水平式「攝影機 App 商店」平台。不是收掉就是乏人問津——服務所有人的平台，最後誰也沒服務到。這片墳場決定了我們「垂直優先」的順序。
Roboflow · Clarifai · Lumeo
Dev-tools: they sell shovels to teams who build their own pipelines. The buyer must still create the meaning. We sell the gold, not the shovel. 開發工具：賣鏟子給自建管線的團隊，買家還是得自己創造意義。我們賣的是金子，不是鏟子。
Butlr
Closest analog — sensing API embedded in third-party SaaS, but occupancy-only (mmWave people-counting). Proves the B2D2B motion works; stops short of vertical meaning. 最接近的對標——嵌入第三方 SaaS 的感測 API，但僅止於人流佔用（毫米波計數）。證明了 B2D2B 模式可行；但止步於垂直意義之前。
IRL API
Embeddable, multi-sensor, vertical-meaning events, sold to software companies. The five-attribute combination no one occupies — in a convergence zone many will enter. Speed and vertical depth decide it. 可嵌入、多感測器、具垂直意義的事件，賣給軟體公司。這個五屬性組合目前無人佔據——但這是條許多人即將湧入的賽道，勝負取決於速度與垂直深度。
